# Study Matchmaker

Study Matchmaker is a full-stack web application designed to help students find compatible study partners, 
manage study sessions, and collaborate more effectively.

---

## Overview

This application allows students to:

- Register and create profiles
- Find study partners based on shared courses
- Send and accept friend requests
- Create and join study groups
- Schedule study sessions
- Track study history
- Set reminders
- Leave reviews after sessions

## How to Use the App

1. Start backend
2. Start frontend
3. Open browser
4. Register account
5. Log in
6. Complete profile
7. Find study matches
8. Send requests or join groups
9. Schedule sessions
10. Leave reviews
